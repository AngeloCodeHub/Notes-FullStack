# Windows ShortCut（快捷鍵、快速鍵）

  >解放滑鼠，增進工作效率，在多個應用程式切換

## 全域

- CTRL + SHIFT + ESC：工作管理員
- Alt + 空白鍵：開啟使用中視窗的快速鍵功能表。

## 視窗的排列控制（雙螢幕與Snap Layout）

	視窗的控制有1.分貼齊螢幕（上下左右）與2.原始視窗，3.最大化視窗，控制的方式不一樣

- Win + 左右箭頭：當前的視窗靠當前螢幕左右，並可選另一邊的切割視窗
- Win + ↑：原始視窗最大化
- Win + ↓：最大化時還原視窗，原始視窗時是最小化
  - Win + Shift + ↓：視窗貼齊或最大化時時，將視窗還原
- Win + D：顯示或隱藏桌面（與Win + M 不同的是可復原）
  - Win + M：將所有打開的視窗最小化（不可復原）
- Win + Alt + 數字鍵：打開釘選在工作列上指定位置應用的右鍵選單
  > 此動作會同時切換到那一個視窗
- Win + Shift + ↑：寬度不變的狀態向螢幕頂端和底部伸展桌面視窗，最大化時沒作用
- Win + Shift + 左右鍵：雙螢幕時將活動視窗切換到不同螢幕
- Win + Space：切換語言，同 Win + Shift + Space

## 其他

- Win + R：Windows Run（可由Windows search "run"）
- Win + V：開啟剪貼簿檢視
- Win +  .：Windows Emoji
- Windows UEFI BIOS無法使用DEL鍵進入（長按shift並重開）
