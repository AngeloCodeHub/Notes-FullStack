在 VS Code 中，您可以使用以下兩種快速方法將一行內容依據分號拆改成多行。我已經先幫您將 [Untitled-1.ps1](cci:7://file:///d:/WorkSpace/Learn-Supabase/Untitled-1.ps1:0:0-0:0) 中的 PATH 路徑拆分好了，方便您查看。

以下是操作教學：

### 方法 1：使用「尋找與取代」 (最快、最精準)
這是處理大量文字的最佳方法。

1. 按下 **`Ctrl + H`** 開啟取代面板。
2. 點擊搜尋框右側的 **「使用規則運算式 (Use Regular Expression)」** 圖示 `.*` (或是按下 `Alt + R`)，確認它已亮起。
3. 在「尋找 (Find)」欄位輸入： **`;\s*`**
   * 解釋：`;` 代表分號，`\s*` 代表分號後可能跟著的空白 (例如 `; `)。
4. 在「取代 (Replace)」欄位輸入： **`\n`**
   * 解釋：`\n` 代表換行符號。
5. 點擊 **「全部取代 (Replace All)」** (或按 `Ctrl + Alt + Enter`)。

### 方法 2：使用多重游標 (直觀)
適合想要手動微調每次換行時使用。

1. 在文件中選取一個分號與其後的空白 (例如 `; `)。
2. 按下 **`Ctrl + Shift + L`**，這會同時選取文件中所有相同的 `; `。
3. 按下 **`Enter`** 鍵，原本的分號就會全部變成換行。
   * (如果您想保留分號並換行，可以按 `Left Arrow` 把游標移到分號後，再按 `Enter`)。

這兩種方法都能大大節省手動修改的時間！
