# Script

- 變數名中只能包含字母和數字。如果變數名稱包含其他字符，則必須用雙引號括起來
- pick（指令）：存取陣列元素（範圍），使用索引提取
- set：修改變數（全域指令），修改規則（path或功能指令）
- 變數宣告不能有特殊字元（如全形字元），只能有英文字母與數字
- key，value的陣列要如何宣告？
→（運算子）：提取陣列或物件（使用key:value）
陣列有分兩種，一種是純陣列，一種是物件陣列
    
    ```powershell
    # 物件陣列範例
    :global aaa {a=1;b=2}
    ```
    
- get是從一條規則取出參數值，find 從參數的值找出item
- 背景執行script指令：execute
